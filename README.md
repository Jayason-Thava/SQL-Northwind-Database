## üß† SQL Concepts Practiced

| Keyword       | Purpose                                                                 |
|---------------|-------------------------------------------------------------------------|
| `SELECT`      | Retrieve specific columns from tables                                   |
| `FROM`        | Specify the source table(s)                                             |
| `WHERE`       | Filter rows based on conditions                                         |
| `IN`, `LIKE`  | Advanced filtering using pattern matching and list inclusion            |
| `COUNT`, `SUM`| Aggregate data for totals and metrics                                   |
| `INNER JOIN`  | Combine rows from related tables based on matching keys                 |
| `GROUP BY`    | Group data for aggregation                                              |
| `ORDER BY`    | Sort results by one or more columns                                     |
